# 🧠 AI SYSTEM INSTRUCTIONS FOR KNOWLEDGE-QUERY-SERVICE
1. **Teknoloji:** Python 3.11+, FastAPI, gRPC, Qdrant, SentenceTransformers.
2. **Write Yasağı (CQRS):** Bu servis SADECE OKUMA (Query) yapar. Qdrant'a vektör ekleme, silme veya güncelleme işlemi yazmak KESİNLİKLE YASAKTIR. (O görev `indexing-service`'e aittir).
3. **Loglama:** Python standart `print()` veya ham `logging.info()` yasaktır. SUTS v4.0 uyumlu JSON basan `structlog` (InterceptHandler ile) kullanılmak ZORUNDADIR.
4. **Thread Safety:** PyTorch/CUDA işlemleri `asyncio.to_thread` ile **bloklamadan** veya OOM patlaması yaratmadan ana thread'de dikkatle (memory limitleri gözetilerek) yönetilmelidir.
